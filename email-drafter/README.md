# Email Drafter

A Claude Code skill that drafts project emails for architecture, engineering, and construction workflows — RFIs, submittals, client updates, consultant coordination, and contractor correspondence with the right tone and structure.

## Install

```bash
# Clone the skills repo
git clone https://github.com/AlpacaLabsLLC/skills.git ~/.claude/skills/email-drafter

# Or copy just this skill
cp -r email-drafter/ ~/.claude/skills/email-drafter/
```

## Usage

Invoke with `/email-drafter` in Claude Code, then describe what you need:

```
/email-drafter
Write an RFI about the unclear flashing detail at the parapet on drawing A-501.
```

```
/email-drafter
Send a submittal review response — revise and resubmit on the porcelain tile sample, colors don't match the approved finish schedule.
```

```
/email-drafter
Client update for the Elm Street renovation — framing is 80% complete, we need the client to pick the kitchen countertop by Friday.
```

```
/email-drafter
Meeting follow-up from today's OAC — attendees were owner, architect, GC, and MEP. We agreed to push the roof demo two weeks. GC to submit revised schedule by Monday.
```

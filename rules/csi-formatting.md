# CSI Formatting — superseded

This studio operates in Western Australia and uses the **NATSPEC** specification system, not CSI MasterFormat.

See [`natspec-formatting.md`](./natspec-formatting.md) for worksection numbering, three-part structure, and AS/NZS reference conventions.

CSI MasterFormat conventions are retained for reference only in case of overseas or US-client work — if required, declare the system being used at the head of the specification document.

To remove this file from the repository:

```
git rm rules/csi-formatting.md
```

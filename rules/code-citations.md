# Code Citations

These conventions apply whenever referencing building codes, standards, or regulations. **Default jurisdiction: Western Australia.**

## Format

- Always cite the **edition year**: "NCC 2022 Vol. 1 D2D5", not "NCC D2D5" or "the building code"
- For NCC clauses, use the current Governing Requirements / Performance Requirement / DTS Provision notation introduced in NCC 2022 (e.g., `NCC 2022 Vol. 1 D2D5`, `NCC 2022 Vol. 2 H1D2`). For pre-2022 references, use the older alphanumeric format with the edition year (e.g., `BCA 2019 D1.6`)
- For Australian Standards: include the full designation and year: `AS 1428.1–2021`, `AS/NZS 3500.1:2021`
- For State legislation: include the year of the principal Act: *Planning and Development Act 2005* (WA); cite specific sections as `s. 162`
- For Local Planning Schemes: cite the LPS number and clause: `City of Fremantle LPS4 cl. 3.2.1`
- For R-Codes: cite the version and clause: `R-Codes Vol. 1 (2024) cl. 5.1.3` or `SPP 7.3 R-Codes Vol. 2 (Apartments) cl. 2.4`

## Common Codes & Standards — Use These Abbreviations

| Code / Document | Full Name |
|------|-----------|
| NCC | National Construction Code (current edition: NCC 2022, with WA-specific variations in NCC Vol. 1 & 2 Appendix WA) |
| BCA | Building Code of Australia — the volumes of the NCC dealing with buildings (Vols. 1 & 2). Use "NCC" in preference unless distinguishing from the PCA |
| PCA | Plumbing Code of Australia — NCC Vol. 3 |
| R-Codes | State Planning Policy 7.3 — Residential Design Codes of Western Australia (Vol. 1: single & grouped dwellings; Vol. 2: apartments) |
| LPS | Local Planning Scheme (e.g., City of Fremantle LPS4, City of Perth LPS26) |
| LPP | Local Planning Policy |
| MRS | Metropolitan Region Scheme |
| SPP | State Planning Policy (e.g., SPP 3.7 Planning in Bushfire Prone Areas) |
| DAP | Development Assessment Panel — JDAP (Joint) or LDAP (Local) per WAPC operational instructions |
| WAPC | Western Australian Planning Commission |
| DPLH | Department of Planning, Lands and Heritage |
| AS / AS/NZS | Australian Standard / joint Australian-New Zealand Standard |
| DDA | *Disability Discrimination Act 1992* (Cth) — implemented in buildings via NCC + AS 1428 series |
| NABERS | National Australian Built Environment Rating System |
| Green Star | Green Building Council of Australia rating tool (cite tool & version: `Green Star Buildings v1`) |
| BAL | Bushfire Attack Level (per AS 3959–2018) |

## WA-Specific Legislation

When a skill output touches planning, building, or heritage matters, cite the relevant Act:

- *Planning and Development Act 2005* (WA)
- *Planning and Development (Local Planning Schemes) Regulations 2015* (WA) — the "deemed provisions" appear in Schedule 2
- *Building Act 2011* (WA) and *Building Regulations 2012* (WA)
- *Architects Act 2004* (WA) — governs use of the title "architect" and registration via the Architects Board of Western Australia (ABWA)
- *Heritage Act 2018* (WA) — State Register of Heritage Places, administered by the Heritage Council of WA / DPLH
- *Strata Titles Act 1985* (WA)
- *Bush Fires Act 1954* (WA)

## Jurisdiction Awareness

- The NCC is adopted nationally but **WA applies state variations** — always check NCC Vols. 1 & 2 *Appendix WA* before relying on a DTS provision
- Planning is a **state and local** responsibility — every site has at least three layers: MRS → LPS → LPP. State the layer when citing
- For development applications, identify the **Responsible Authority**: usually the local government, sometimes WAPC, sometimes a JDAP (development cost > $2M residential / > $20M general triggers JDAP referral per the *Planning and Development (DAP) Regulations 2011*)
- Never state "this complies with the NCC" — say "this appears consistent with NCC 2022 Vol. 1 D2D5 (subject to BAL assessment / engineering certification / etc.)"
- Flag when requirements **vary by local government**, when the applicable LPS amendment is uncertain, or when an LPP overrides a deemed provision

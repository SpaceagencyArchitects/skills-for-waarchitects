# NATSPEC Formatting

These conventions apply whenever referencing or writing construction specifications. **Default specification system: NATSPEC** — the Australian national specification system administered by Construction Information Systems Limited.

## Worksection References

- Use the **NATSPEC worksection number and title**: `0181 Concrete in situ`, `0411 Brick and block construction`
- Worksection numbers are 4-digit codes; cite with the title after a single space (no dash, no em-dash)
- For NATSPEC Domestic worksections (residential), prefix with the domestic series: `D0411 Brick and block construction`
- Always state which NATSPEC product is being used: NATSPEC Building, NATSPEC Domestic, NATSPEC Interiors, or a custom in-house spec derived from NATSPEC
- For practices using a different system (e.g., ArchiSpec, an in-house template), name the system on first mention and apply the same three-part structure

## Worksection Groupings (NATSPEC Building, indicative)

NATSPEC organises worksections into groups roughly aligned to trade/element. The exact numbering is subject to revision — always confirm against the current NATSPEC reference manual. Indicative groups:

| Group | Subject |
|-------|---------|
| 01xx | General — preliminaries, quality, contract administration |
| 02xx | Sitework, demolition, earthworks, site services |
| 03xx | Structural engineering systems |
| 04xx | Masonry |
| 05xx | Metalwork and structural steel |
| 06xx | Carpentry, joinery and timber |
| 07xx | Thermal, moisture and waterproofing |
| 08xx | Doors, windows and glazing |
| 09xx | Linings, finishes, painting |
| 10xx, 11xx | Specialties, equipment, fixtures |
| 13xx | Special construction (e.g., pools, lifts, modular) |
| 14xx | Conveying systems |
| 20xx–27xx | Mechanical, hydraulic, fire, electrical, communications services |

Do not invent worksection numbers — if uncertain, cite the worksection title only and flag the number as `[confirm NATSPEC ref]`.

## Three-Part Worksection Structure

NATSPEC worksections follow a three-part structure that aligns with international convention:

- **1 General** — references, interpretation, submissions, inspections, quality requirements, samples
- **2 Products** — materials, components, manufacture, quality of materials, alternatives
- **3 Execution** — preparation, installation, application, finishing, completion, defects liability

## Australian Standards within Specifications

- Reference the relevant **AS / AS/NZS** standard with the year: `to AS 3600–2018` or `complying with AS/NZS 1170.0:2002`
- For products with branded equivalents, name the AS first, then list acceptable manufacturers/products
- For NCC-driven requirements (fire, structural, accessibility), cite the NCC clause that drives the AS reference: `provide fire resisting construction to NCC 2022 Vol. 1 Spec C1.1, tested to AS 1530.4`

## Cross-References

- Reference related worksections by full number and title: "refer to worksection 0731 Membranes"
- Never use vague references like "see waterproofing section" — always cite the specific worksection
- For cross-references to drawings, use sheet number and detail reference: "refer to detail 3/A6.04"

## When NATSPEC is not the source

- For overseas projects, US-client work, or where the contract specifies CSI MasterFormat, use the original CSI three-part 6-digit format (`09 29 00 — Gypsum Board`) and state the system being used at the head of the spec
- Never mix systems within a single specification document
